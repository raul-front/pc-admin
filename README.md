# admin
后台管理

# 初始化项目
1. 拉取代码
```
git clone git@github.com:raul-front/pc-admin.git
cd pc-admin
```
2. 拉取子仓库代码
```
make initModule
```
3. 安装依赖
```
npm install
```

# 运行
1. 开发环境
```
npm run dev
```

2. 打包测试环境
```
npm run build:test
```

3. 打包生产环境
```
npm run build
```
